# 도메인 용어집

이 문서는 게시판 도메인에서 사용하는 용어를 정의한다. 기능 구현 전후로 용어가 추가되면 이 파일을 갱신한다.

## 현재 기준 용어

| 용어 | 영문명 | 설명 |
| --- | --- | --- |
| 게시글 | Post | 사용자가 작성한 게시판의 기본 콘텐츠 단위 |
| 제목 | Title | 게시글을 요약하는 짧은 문구 |
| 본문 | Content | 게시글의 주요 내용 |
| 작성자 | Author | 게시글을 작성한 사용자 또는 주체 |
| 댓글 | Comment | 게시글에 대해 작성하는 하위 의견 |
| 첨부파일 | Attachment | 게시글 또는 댓글에 연결되는 파일 |
| 조회수 | View Count | 게시글이 조회된 횟수 |
| 좋아요 | Like | 사용자가 게시글에 긍정 반응을 남기는 행위 |
| 게시판 | Board | 게시글을 분류하거나 모아두는 공간 |

## 용어 관리 규칙

- 기능 명세에서 새로운 도메인 용어를 사용하면 이 파일에 추가한다.
- 같은 개념에 여러 이름을 사용하지 않는다.
- 화면 표현과 도메인 용어가 다를 경우 차이를 명확히 적는다.
- 구현 클래스명은 가능한 한 이 용어집의 영문명을 따른다.

## 미확정 용어

아래 항목은 기능 명세 작성 시 확정한다.

- 회원과 작성자의 관계
- 게시판 분류 정책
- 게시글 공개 범위
- 댓글과 대댓글 지원 여부
- 첨부파일 정책
