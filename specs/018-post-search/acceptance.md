# 018 게시글 검색 인수 조건

## 문서 완료 조건

- `GET /posts?keyword=...` 검색 조건이 문서화되어 있다.
- 검색 대상이 게시글 제목과 본문이라고 명시되어 있다.
- 검색 결과가 기존 목록 응답 구조와 같다고 명시되어 있다.
- 삭제된 게시글은 검색 결과에서 제외된다고 명시되어 있다.
- 공백 검색어는 전체 목록 조회와 동일하게 처리한다고 명시되어 있다.

## 기능 인수 조건

- `keyword`가 제목에 포함된 게시글은 검색 결과에 포함된다.
- `keyword`가 본문에 포함된 게시글은 검색 결과에 포함된다.
- 검색은 대소문자를 구분하지 않는다.
- 삭제된 게시글은 검색 결과에 포함되지 않는다.
- 검색 결과는 생성 시각 기준 최신순으로 정렬된다.
- 검색 결과 항목은 댓글 수 `commentCount`를 포함한다.
- 검색 결과가 없으면 빈 `posts` 목록과 페이징 메타데이터를 반환한다.
- `keyword`가 없거나 공백뿐이면 기존 전체 목록 조회와 동일하게 동작한다.
- 잘못된 `page` 또는 `size`는 `400 Bad Request`, `code=INVALID_REQUEST`를 반환한다.
- 가능한 경우 `.\gradlew.bat test`가 성공한다.
