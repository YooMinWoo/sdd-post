# 001 게시글 작성 인수 조건

## 성공 조건

- 유효한 제목, 본문, 작성자로 `POST /posts`를 호출하면 `201 Created`를 반환한다.
- 응답에는 생성된 게시글 id, 제목, 본문, 작성자, 생성 시각이 포함된다.
- 저장된 게시글은 JPA 어댑터를 통해 영속화될 수 있다.

## 잘못된 입력 조건

- 빈 제목은 `400 Bad Request`를 반환한다.
- 100자를 초과한 제목은 `400 Bad Request`를 반환한다.
- 빈 본문은 `400 Bad Request`를 반환한다.
- 5,000자를 초과한 본문은 `400 Bad Request`를 반환한다.
- 빈 작성자는 `400 Bad Request`를 반환한다.
- 50자를 초과한 작성자는 `400 Bad Request`를 반환한다.

## 완료 기준

- 도메인, 애플리케이션, Web 어댑터, Persistence 어댑터 테스트가 기능을 검증한다.
- `.\gradlew.bat test`가 통과하거나 실패 사유가 기록된다.
